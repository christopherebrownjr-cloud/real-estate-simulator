# New Realtor Simulator

A web-based real-estate career simulator. The project is being developed in controlled milestones so that gameplay rules are approved before they become code.

## Development status

Milestone 0 foundation in progress. The current application shell intentionally contains no unapproved simulation rules.

## Local development

```text
npm install
npm run dev
```

The app is designed for static deployment to GitHub Pages. The production build is created with `npm run build`.
