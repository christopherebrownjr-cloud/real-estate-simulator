# Open Product Questions

These are the smallest decisions that currently block simulation coding:

1. Is the first release web-only and single-player?
2. Should saves be browser-local for the MVP, or must players use an online account/backend?
3. Which exact actions advance executive-controlled time, and what time cost does each action have?
4. What exact conditions convert a lead into a client and allow a deal to close?
5. Should the draft trust, reputation, lead-score, and commission values be approved as written, or revised?
6. Which screens and player flows belong in First Playable?

No implementation should assume answers to these questions.
